# 上海邮政揽投场地工艺设备智能配置平台

基于GLM-4.5-Flash大模型的邮政揽投场地工艺设备智能配置系统。

## 功能概述

本平台基于《上海邮政揽投生产场地工艺设备应用的前瞻性研究》报告中的综合评分模型，为投资者和管理者提供智能化的设备配置方案。

### 核心功能

1. **综合评分模型** - 基于四大维度（场地条件60%、业务压力25%、人工压力5%、自动化适配10%）的量化评分
2. **设备配置模板** - 根据评分自动生成1-2套设备配置参考方案
3. **AI智能报告** - 调用智谱GLM-4.5-Flash大模型生成详细的专业分析报告
4. **投资效益分析** - 自动测算投资成本、运营费用和回收期

### 评分模型

| 维度 | 权重 | 指标 | 标准化方法 |
|------|------|------|-----------|
| 场地条件 | 60% | 操作区域面积 | Sigmoid非线性 |
| 业务压力 | 25% | 日均处理量/峰值量/高峰天数 | Min-Max线性 |
| 人工压力 | 5% | 人均处理量/场地利用率 | Sigmoid非线性 |
| 自动化适配 | 10% | 可上机率/分拣道段数 | Sigmoid非线性 |

### 设备类型

平台涵盖7类分拣设备：
- 模组带分拣机（首选推荐）
- 直线交叉带分拣机
- 环形翻盘分拣机
- 小型直线窄带分拣机
- 圆盘分拣机
- 往复式分拣机
- 三段式基础设备

## 技术栈

- 前端：HTML5 + CSS3 + 原生JavaScript
- AI引擎：智谱GLM-4.5-Flash
- 部署：GitHub Pages

## 使用方法

1. 在左侧表单输入场地基础参数
2. 点击"智能分析与方案生成"按钮
3. 查看综合评分结果和设备配置模板
4. 点击"生成AI详细报告"获取专业分析报告

## 数据来源

所有设备参数、评分模型权重、案例数据均来自《上海邮政揽投生产场地工艺设备应用的前瞻性研究》报告。
